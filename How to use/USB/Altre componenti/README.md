# Altre Componenti nella Cartella USB

Questi componenti non appartengono a una sottocartella specifica ma sono inclusi direttamente nella cartella principale USB.

### • BarCode Scanner E

Emulatore HID che invia stringhe come se fossero codici a barre digitati da una tastiera.

Funzionalità ampliate:

- Simulazione lettore barcode USB.
- Output immediato delle stringhe tramite HID.
- Utile per test di sistemi di cassa, inventario e automazione dati.

Esempio pratico:

- Configurare il campo di input su PC.
- Scansionare un codice dal Flipper → il testo viene digitato automaticamente.

### • Clippy

Assistente USB divertente che mostra popup o animazioni tramite un host supportato.

Funzionalità ampliate:

- Messaggi animati stile “Clippy”.
- Attivazione da USB come dispositivo dedicato.

Esempio pratico:

- Collegare il Flipper → comparsa automatica di messaggi ironici stile anni 2000.

### • Flip TDI

Interfaccia USB per comunicazioni TDI e debug specifici.

Funzionalità ampliate:

- Connessione a dispositivi su bus TDI.
- Lettura/scrittura dati e test.

Esempio pratico:

- Connettere dispositivo con TDI → inviare comandi diagnostici.

### • HID File Transfer

Trasferimento file tramite protocollo HID, senza attivare la modalità mass storage.

Funzionalità ampliate:

- Trasferimento sicuro senza mounting di drive.
- Compatibile con tool PC dedicati.

Esempio pratico:

- Avviare software lato PC → inviare file dal Flipper tramite HID.

### • Lego Dimensions Toy Pad

Emulatore del Toy Pad USB per il gioco Lego Dimensions.

Funzionalità ampliate:

- Emulazione basi e personaggi.
- Supporto a scenari e colori pad.

Esempio pratico:

- Selezionare personaggio → Flipper lo emula come Toy Pad originale.

### • Mass Storage

Il Flipper si presenta come una chiavetta USB tradizionale.

Funzionalità ampliate:

- File system esposto direttamente al PC.
- Trasferimenti rapidi.

Esempio pratico:

- Attivare Mass Storage → copiare file da/verso PC.

### • Mouse Jiggler

Simula movimenti mouse per evitare standby o blocco schermo.

Funzionalità ampliate:

- Movimenti casuali o regolari.
- Compatibile con qualunque OS.

Esempio pratico:

- Attivarlo durante presentazioni → il PC non va in sleep.

### • MTP

Media Transfer Protocol per accedere ai file come dispositivo multimediale.

Funzionalità ampliate:

- Accesso a foto, file e media.
- Compatibile con Windows/macOS/Linux.

Esempio pratico:

- Collegare Flipper → visualizzazione contenuti via MTP.

### • Portal of Flipper

Modulo creativo USB usato da giochi o app interne.

Funzionalità ampliate:

- Interazioni USB custom.
- Impiegato da minigiochi o app sperimentali.

Esempio pratico:

- Avviare app dedicata sul Flipper → interazione tramite USB.

### • U2F

Il Flipper funge da chiave di sicurezza U2F.

Funzionalità ampliate:

- Login 2FA su siti U2F.
- Sicurezza hardware.

Esempio pratico:

- Associare Flipper come chiave → toccare il pulsante per approvare.

### • USB Consumer Control

Invia comandi multimediali tramite HID.

Funzionalità ampliate:

- Volume su/giù, play, pausa, mute.
- Controllo semplice e rapido da USB.

Esempio pratico:

- Usarlo come telecomando multimediale del PC.

### • USB HID Autofire

Premi ripetuti automatici per gaming e automazione.

Funzionalità ampliate:

- Pressione tasti ad alta frequenza.
- Configurazione intervalli.

Esempio pratico:

- Impostare autofire su un tasto → utile per test o videogiochi.

### • USB Remote

Controllo tastiera e mouse tramite interfaccia grafica.

Funzionalità ampliate:

- Movimento mouse via Flipper.
- Invio tasti rapidi.

Esempio pratico:

- Usarlo come telecomando USB per un PC su proiettore.

### • USB MIDI

Il Flipper diventa uno strumento MIDI.

Funzionalità ampliate:

- Invio note MIDI.
- Compatibile con synth, DAW e software musicali.

Esempio pratico:

- Suonare note dal Flipper → controllare un sintetizzatore virtuale.

### • Xbox360 USB Game Controller

Emulazione controller Xbox360 via USB.

Funzionalità ampliate:

- Full HID XInput.
- Compatibile con giochi PC.

Esempio pratico:

- Collegare il Flipper → rilevato come controller Xbox per gaming rapido.
